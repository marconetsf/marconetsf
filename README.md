## Marcone Tenorio da Silva Filho

**IoT Developer | Embedded Systems | Mobile & Web**

Desenvolvedor focado em IoT e sistemas embarcados, criando middleware que integra sistemas de TI com sistemas mecatronicos. Experiencia em firmware, automacao residencial/industrial e desenvolvimento mobile com Flutter.

---

### Areas de Atuacao

**IoT & Sistemas Embarcados** — Firmware em C/C++, comunicacao LoRaWAN, gerenciamento WiFi (ESP-IDF), automacao residencial e industrial (OpenPLC).

**Desenvolvimento Mobile** — Aplicacoes com Flutter/Dart, integracao com Firebase (Auth, Firestore), UI responsiva.

**Web & Outros** — React, TypeScript, JavaScript, Python (OpenCV), ferramentas de produtividade.
